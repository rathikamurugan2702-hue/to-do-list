# To-Do List Application

A functional and interactive **To-Do List** web application. This project helps users manage their daily tasks efficiently with a clean, user-friendly interface.

## 🚀 Features
* **Add Tasks:** Quickly add new items to your list.
* **Mark as Done:** Toggle tasks as completed to track progress.
* **Delete Tasks:** Remove tasks you no longer need.
* **Persistent Storage:** (Optional - include if you used LocalStorage) Saves your tasks even after refreshing the browser.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure for the task input and list.
* **CSS3:** Custom styling for a modern, responsive look.
* **JavaScript:** DOM manipulation to handle task logic and events.

## 📂 Installation & Usage
1. Download or clone this repository.
2. Open the `index.html` file in any web browser.
3. Start adding your tasks!
